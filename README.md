## Projetos

### [SpoDop](https://github.com/LuisG-santos/spodop-api) — Plataforma SaaS para o Agronegócio
> Em desenvolvimento ativo

Plataforma web para gestão de propriedades rurais — monitoramento climático, controle de aplicações de defensivos e assistente de IA planejado.

Decisões técnicas relevantes:
- API REST dedicada com **Node.js + Express**, separada do frontend — pensando em reaproveitamento quando o app mobile for desenvolvido
- Arquitetura em camadas: **Controllers → Use Cases → Repositories**, com injeção de dependência via factory functions
- Modelagem relacional com **Prisma ORM + PostgreSQL**: entidades de Propriedade, Talhão, Aplicação e Usuário com FKs e campos nullable justificados tecnicamente
- Auth com **JWT + bcrypt**, proteção contra user enumeration, validações com `validator.js`

### [Belivio](https://github.com/LuisG-santos/belivio) — SaaS de Agendamentos
> Concluído · Deploy em produção com domínio personalizado

Plataforma de agendamentos com arquitetura multi-tenant — desenvolvimento fullstack completo, do banco ao deploy.

- Arquitetura **multi-tenant** com isolamento de dados por tenant
- Backend com **API Routes (Next.js)**, lógica de agendamento e regras de negócio
- Modelagem e manipulação de dados com **Prisma + PostgreSQL**
- Deploy em produção na **Vercel** com domínio personalizado
